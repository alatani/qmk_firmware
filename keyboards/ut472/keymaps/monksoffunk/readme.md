# UT47.2 monksoffunk keymap
```
This is mainly for Japanese IME users.
Switch Mac OSX / Windows mode on L3 layer. 
L3 has also Kana / Eisu.
```

## default layer
```
     -------------------------------------------------------------------------.
    | Esc |  Q  |  W  |  E  |  R  |  T  |  Y  |  U  |  I  |  O  |  P  |Bspace |
    |-------------------------------------------------------------------------+
    |Tab/Ctl|  A  |  S  |  D  |  F  |  G  |  H  |  J  |  K  |  L  |  ;  | Ent |
    |-------------------------------------------------------------------------+
    | Shift |  Z  |  X  |  C  |  V  |  B  |  N  |  M  |  ,  |  .  |  /  |Sh/En|
    |-------------------------------------------------------------------------+
    | Ctrl| L3 | Alt | Gui  |  L2  |   Space   |  L1  | Left| Down|  Up |Right|
    `-------------------------------------------------------------------------'
```

## L1 / Raise layer
```
    ,-------------------------------------------------------------------------.
    | ` ~ |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |  9  |  0  | Delete|
    |-------------------------------------------------------------------------+
    |      |     |     |     |     |     |     |  -  |  =  |  [  |  ]  |  "   |
    |-------------------------------------------------------------------------+
    |       |     |     |     |     |     |     |     |     |     |  \  |     |
    |-------------------------------------------------------------------------+
    |     |     |     |Capsl|      |          |       | Home| PgDn| PgUp| End |
    `-------------------------------------------------------------------------'
```
